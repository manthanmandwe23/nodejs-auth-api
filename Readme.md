# Node.js Backend API Project

A backend-focused project demonstrating real-world API development and authentication system.

 Features
- User Registration & Login
- JWT Authentication
- File Upload (Avatar & Cover Image)
- Secure Password Hashing (bcrypt)
- RESTful API Architecture

 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT

 Project Structure
- controllers
- db
- middlewares
- models
- routes
- utils
